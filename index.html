Sitemap Header
Intro
Overview
;; In FASTN.ftd
-- fastn.dependency:fastn-community.github.io/navigational-sitemap-header

;; In the file where you want to add the header (e.g., `index.ftd`)

;; Add this at the top of the file
-- import: fastn-community.github.io/navigational-sitemap-header as header

;; Add this where you want to show the component
-- header.header:
site-logo: $assets.files.images.ipsum-logo.svg
site-url: index.html
Lang:
ftd
Properties
`site-name`
{
type:
Caption (`optional`)
}
It's used to define the site name for your header component.
`site-image`
{
type:
ftd.image-src (`optional`)
}
It's used to define the site image for your header component.
`site-url`
{
type:
String (`required`)
}
It's used to define the site url for your header component.
